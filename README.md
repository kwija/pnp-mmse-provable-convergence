# Convergence prouvée de PnP-ISTA avec des débruiteurs MMSE

**Auteur :** KHOUAJA Ahmed  
**Formation :** Master 2 MIASHS — Université Lyon 2  
**Cours :** Problèmes Inverses (Nelly Pustelnik)

## Description

Analyse de l'article *"Provable Convergence of Plug-and-Play Priors with MMSE Denoisers"* (Xu et al., IEEE Signal Processing Letters, 2020).

Le notebook présente :
- Le problème inverse traité et le type de dégradation
- La méthode PnP-ISTA et son fonctionnement
- La contribution théorique (convergence avec débruiteur MMSE)
- Une reproduction partielle des résultats avec DeepInverse
